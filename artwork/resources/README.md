# Artwork Resources

This directory contains resource images used by the artwork XML files.

## Required Images

The following PNG images should be placed in `~/.skyscraper/resources/`:

- `miyoo-gradient.png` - A gradient overlay for the top of screenshots (250x100 pixels)
- `miyoo-bg.png` - Background image for style 2 (250x360 pixels)
- `miyoo-fade-left.png` - Gradient fade for left edge transparency (250x360 pixels)

## Creating Your Own Resources

You can create these images using any image editor:

1. **miyoo-gradient.png**: Create a 250x100 image with a black-to-transparent gradient from top to bottom
2. **miyoo-bg.png**: Create a 250x360 solid color or themed background
3. **miyoo-fade-left.png**: Create a 250x360 image with white-to-transparent gradient from left to right

## Default Colors

For OnionOS themes, common background colors are:
- Default dark theme: `#1a1a2e` (dark blue-gray)
- Light theme: `#f5f5f5`
